Perpetual Order Scheduler

自动下单机器人

支持普通定时任务和量化策略两类配置。量化策略也可以通过 API 写入 strategy_tasks。

配置中心

量化策略逻辑：初始化读取近 3 根已收盘 1 分钟 K 线，选涨幅最大的那根，按 (最高价 - 最低价) / 2 计算 use_close_price。后续每分钟若收跌则按 min(下单金额, 合约账户余额) 市价做空，成交后挂限价止盈，10 分钟未平仓则强制市价平仓。
